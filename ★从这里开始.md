# 从这里开始 👋

欢迎接手 **FCN Watchlist 网站**。有两种上手方式：

### 方式一（推荐，省事）：让 Claude Code 帮你部署
1. 先自己注册一个 GitHub 账号（这步必须人来做）。
2. 在本文件夹里打开终端运行 `claude`，直接说：**「帮我部署这个网站」**。
   它会自动读取本目录的 `CLAUDE.md` 运行手册，一步步带你装环境、建仓库、上线。
   遇到只能人做的步骤（注册账号、浏览器授权登录、向 Dave 要 DeepSeek key），它会停下来让你做。

### 方式二：自己照文档操作（不需要会编程）
按顺序看这 3 份：

| 顺序 | 文档 | 什么时候看 |
|------|------|-----------|
| 1️⃣ | **部署到你自己的GitHub.md** | **最先做**：注册 GitHub、把网站挂到你自己账号下（一次性，约 20 分钟） |
| 2️⃣ | **交接使用手册.md** | 部署完之后：装环境、配 key、以及**每周怎么更新**（日常就看这份） |
| 3️⃣ | **代码逻辑说明.md** | 想了解程序内部怎么跑、想改东西时再看 |

## 一句话总览
这是一个每周更新的网站：你**每周换 3 个 Excel**（低波精选 / 高波精选 / 市场热度，即 `低波精选组` / `高波精选组` / `市场热度榜` 三个文件），
**双击 `更新网站.bat`**，程序自动拉行情 + AI 写分析，生成网站数据并推送上线。

## 还需要找 Dave 要 2 样东西
1. **DeepSeek API key**（AI 分析必需）→ 放进你自己新建的 `deepseek_key.txt`
2.（可选）富途账号自己注册，装 OpenD 用于分析师评级数据

有任何问题找 Dave
