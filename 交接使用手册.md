# FCN Watchlist 交接使用手册

> 这是一个「每周更新的 FCN 标的网站」。整个网站的内容由一个文件 `watchlist.json` 驱动，
> 而 `watchlist.json` 由 `generate_watchlist.py` 根据 **3 个 Excel** 自动生成（拉行情 + AI 写分析）。
>
> **每周更新 = 换掉 3 个 Excel → 双击「更新网站.bat」→ 等约 25 分钟 → 自动上线。**

有问题随时找 Dave（daveliao999@gmail.com）。

## 0. 这个文件夹里有什么

| `低波精选组.xlsx` / `高波精选组.xlsx` / `市场热度榜.xlsx` | **每周你要替换的 3 个输入榜单** | **每周换** |

| `fill_terms.py` | 把反解结果写进 3 个榜单 Excel + 统一代码格式 | 见「二·0」 |
| `desk_quotes.csv` | 票息↔IV 标定集（反解引擎依赖，**勿删**；报价环境变了补数据重跑） | 偶尔维护 |

| `requirements.txt` | Python 依赖清单 | 不用动 |
| `更新网站.bat` | 一键更新（拉代码→生成→推送） | 不用动 |
| `deepseek_key.txt` | DeepSeek API key（**需你自己新建**，见下） | 首次创建 |

> **`部署到你自己的GitHub.md`**。本手册默认你已经按那份做完了一次性部署。

## 一、首次配置（每人只做一次，约 20 分钟）

### 1. 装两个软件
- **Python 3.10+**：https://www.python.org/downloads/ —— 安装时务必勾选 **"Add Python to PATH"**
- **Git**：https://git-scm.com/download/win —— 一路默认下一步即可

### 2. 装 Python 依赖
打开「命令提示符」（Win 键搜 `cmd`），把本文件夹拖进窗口前先 `cd` 进去，例如：
```
cd %USERPROFILE%\Desktop\FCN_Watchlist_交接
pip install -r requirements.txt
```

### 3. 配置 DeepSeek key（AI 分析必需）
向 Dave 索取 DeepSeek API key，在本文件夹里新建一个文本文件 **`deepseek_key.txt`**，
把 key 粘贴进去保存（**只有一行，没有引号、没有空格**）。

> ⚠️ `deepseek_key.txt` 已被 `.gitignore` 排除，**绝对不要**把 key 写进任何会提交到 GitHub 的文件——仓库是公开的。
> key 也可以改用环境变量 `DEEPSEEK_API_KEY`，效果一样。

### 4.（可选，推荐）装富途 OpenD
分析报告里的「分析师评级 / 目标价 / 财报」来自富途 OpenAPI。**没有它脚本也能跑完**，只是少这一个维度。
- 下载图形界面版 Futu OpenD：https://www.futunn.com/download/OpenAPI
- 用**你自己**的富途账号登录（不要和别人共用，会互相挤下线）
- 跑脚本前保持 OpenD 开着并已登录即可

### 5. 完成 GitHub 部署
见 `部署到你自己的GitHub.md`。做完后这个文件夹会变成一个连着你 GitHub 仓库的目录，
以后双击 `更新网站.bat` 就能推送上线。

## 二、每周更新流程（约 30 分钟，其中 25 分钟是等 AI）

### 0.（询价前置）用 AI 反解执行价 / 敲入价——本轮新增

三个榜单的**执行价/敲入价现在由 AI 按 IV 反解**，不再手填；**票息**始终来自交易台真实询价。顺序：

1. 建 `lists_input.xlsx`（两列：**组别 / 代码**，IV 留空）→ `python fetch_iv.py`
   （用 OpenD 自动拉每只「90% 现价档、6 个月 put」的 IV，港股无期权退 6M HV；**需 OpenD 开着并登录**）
2. `python strike_advisor_pipeline.py` → 输出 `建议结构.csv`（执行价/敲入价，整数，含 **85% 硬顶**规则）
3. `python fill_terms.py` → 自动把执行价/敲入价写进下面 3 个榜单 Excel，并把代码统一成 `NVDA US / 9988 HK`
4. 把 3 个榜单 Excel 发**交易台询价** → 真实票息填回「票息」列 → 再走下面第 1 步起

> 反解规则细节（IV 分档区间：30-70→票息15-25%、70-100→25-30%；执行价硬顶 85%；`desk_quotes.csv` 重标定）
> 见同目录 **`团队操作手册.md` 第三节**。

### 1. 更新 3 个榜单 Excel（文件名不能改）

- `低波精选组.xlsx` — 网站显示「**低波精选**」，约 20 只
- `高波精选组.xlsx` — 网站显示「**高波精选**」，约 20 只
- `市场热度榜.xlsx` — 网站显示「**市场热度型**」，约 20 只

   **列格式**：`区域 / 代码 / 名称 / 行业 / 执行价 / 敲入价 / 敲入类型 / 敲出价 / 票息`
   - `区域`：写「港股 / 美股」，脚本看是不是「港」开头判断市场
   - `代码`：`NVDA US` / `9988 HK`，或纯代码 `NVDA`/`700` 均可（市场以「区域」列为准，港股不用补 0）
   - 执行价/敲入价由上面「0」自动填；**票息**填交易台真实报价；均写**小数**（`0.8` = 80%）
   - 同一标的出现在多个榜单 = 自动合并成「多类型」标签，不会重复

2. **（可选）打开并登录富途 OpenD**（想要分析师数据就开）

3. **双击 `更新网站.bat`**，等窗口出现 ✅ 完成。
   它会依次：拉取最新代码 → 生成分析（约 25 分钟，别关窗口）→ 提交 → 推送上线。

4. 几分钟后打开你的网站（`https://你的用户名.github.io`）检查：
   - 三个策略卡片各约 20 只标的
   - 随手点开两三只看详情页分析是否完整

## 三、出错怎么办

| 现象 | 处理 |
|------|------|
| 窗口显示 ❌ | 截图整个窗口发给 Dave |
| 提示「未找到 DeepSeek key」/ 程序直接退出 | 检查 `deepseek_key.txt` 是否在本文件夹、内容是否只有 key 一行 |
| 提示「Futu 连接失败」| 不影响生成，想要分析师数据就先开并登录 OpenD 再重跑 |
| 个别标的分析不完整（详情页缺段落） | 跑 `python repair_watchlist.py`（自动检测并单独修复），完成后：`git add watchlist.json` → `git commit -m "repair"` → `git push` |
| `git push` 报 conflict / rejected | 先 `git pull --rebase` 再重新双击 bat；解决不了找 Dave |
| 网站打开是空白 / 数据没更新 | 等 1–2 分钟让 GitHub Pages 重建；强制刷新（Ctrl+F5）；确认 `git push` 成功了 |

## 四、只想改内容、不重跑 AI（小修改）

网站右下角有个 **ADMIN** 按钮（密码 `olive2026`，可在 `index.html` 里搜 `ADMIN_PASSWORD` 修改）。
进去可以改条款 / 改本周精选 → 点「导出 watchlist.json」下载 → 用下载的文件覆盖本文件夹里的 `watchlist.json` → 命令行执行：
```
git add watchlist.json
git commit -m "manual edit"
git push
```
1–2 分钟后网站更新。
